Projects

Accounting Ticketing System

Created a program that managed support tickets in C. Validation logic was implemented to verify a user before giving access to ticket information. Authenticated users can view, open, close, create, and display ticket messages, as well as, add new customer or agent accounts. Records of tickets and accounts are loaded from a text file and changes made during the session are saved by overwriting the old text file.



Shipping Handling System

Created a program in C++ that managed items that needed to be shipped. Each item was an object of a class containing the quantity on hand, quantity needed, description, and expiry date. Accessing the program allowed for users to view the items, update quantity of items, remove items, sort the items, and display detailed descriptions of each item. The record of items can are loaded from an external file and on exit will automatically remove items that are ready to be shipped from the records and generate a summary of the shipped items.
